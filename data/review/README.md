# review

검증용 중간 산출물을 저장하는 폴더입니다.

현재 예시:

- [mismatch_candidates_2021_2025.jsonl](/e:/Project/tax_exam3/data/review/mismatch_candidates_2021_2025.jsonl)
  - 풀이 답안과 배포답안 불일치 후보 목록

용도:

- LLM 검증 배치 입력
- 수동 검토 대상 추출
- 데이터 보정 전후 비교

주의:

- 운영 서비스에서 직접 읽는 폴더는 아닙니다.
- 재생성 가능한 중간 파일 위주라, 필요 시 삭제 후 다시 만들 수 있습니다.
