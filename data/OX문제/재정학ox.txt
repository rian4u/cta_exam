### [제1파트] 재정학의 기초 및 후생경제학 OX 문제 100선

1. 재정학은 정부의 경제적 행위와 그것이 자원배분 및 소득분배에 미치는 영향을 연구하는 학문이다. / 답: O / 설명: 공공부문의 경제적 역할을 분석하는 것이 재정학의 핵심입니다.
2. 유기체적 국가관에 따르면 국가는 개인들의 단순한 집합체에 불과하다. / 답: X / 설명: 유기체적 국가관은 국가를 개인과는 별개의 독자적인 인격과 목적을 가진 실체로 봅니다.
3. 기계론적 국가관은 국가를 개인의 목적 달성을 위한 수단으로 간주한다. / 답: O / 설명: 개인주의에 기초하여 국가의 존재 이유를 개인의 후생 증진에서 찾습니다.
4. 재정의 자원배분 기능은 소득 불평등을 시정하는 것을 주된 목적으로 한다. / 답: X / 설명: 자원배분 기능은 효율성 제고를, 소득재분배 기능이 형평성 시정을 목적으로 합니다.
5. 파레토 개선이란 다른 사람의 후생을 감소시키지 않으면서 최소한 한 사람 이상의 후생을 증가시키는 것을 말한다. / 답: O / 설명: 사회 전체의 후생이 순증가하는 변화를 의미합니다.
6. 파레토 효율적인 상태에서는 더 이상의 파레토 개선이 불가능하다. / 답: O / 설명: 모든 자원이 효율적으로 배분되어 누군가의 후생을 줄이지 않고는 다른 이의 후생을 높일 수 없는 상태입니다.
7. 후생경제학 제1정리는 모든 소비자의 선호가 강단조성을 가질 때 경쟁시장 균형은 파레토 효율적임을 의미한다. / 답: O / 설명: 시장 기구의 효율성을 옹호하는 이론적 근거입니다.
8. 후생경제학 제2정리에 따르면 초기 부문의 적절한 재분배가 있다면 임의의 파레토 효율적 배분을 시장 균형으로 달성할 수 있다. / 답: O / 설명: 효율성과 형평성을 분리하여 달성할 수 있음을 시사합니다.
9. 엣지워스 상자(Edgeworth Box)에서 계약곡선 상의 모든 점은 파레토 효율적이다. / 답: O / 설명: 계약곡선은 두 소비자의 무차별곡선이 접하는 점(MRS가 일치하는 점)들의 궤적입니다.
10. 소비의 파레토 효율성 조건은 두 소비자의 한계대체율(MRS)이 일치하는 것이다. / 답: O / 설명: $MRS_{xy}^A = MRS_{xy}^B$일 때 소비의 효율성이 달성됩니다.
11. 생산의 파레토 효율성 조건은 두 생산요소의 한계기술대체율(MRTS)이 일치하는 것이다. / 답: O / 설명: $MRTS_{lk}^X = MRTS_{lk}^Y$일 때 요소 투입의 효율성이 달성됩니다.
12. 종합적 효율성(생산과 소비의 결합)은 MRS와 MRT(한계변환율)가 일치할 때 달성된다. / 답: O / 설명: 주관적 교환비율(MRS)과 기술적 변환비율(MRT)이 같아야 합니다.
13. 한계대체율(MRS)은 무차별곡선의 기울기(절댓값)를 의미한다. / 답: O / 설명: 소비자가 주관적으로 느끼는 두 재화 사이의 교환비율입니다.
14. 한계변환율(MRT)은 생산가능곡선의 기울기를 의미하며, 두 재화의 한계생산비 비율(MCx/MCy)과 같다. / 답: O / 설명: 생산 측면에서의 기회비용을 나타냅니다.
15. 보상변동(CV)은 가격 변화 후의 효용을 변화 전의 효용으로 되돌리기 위해 필요한 소득의 변화다. / 답: O / 설명: 새로운 가격 체계하에서 이전 효용을 유지하기 위한 보상액을 측정합니다.
16. 대등변동(EV)은 가격 변화 전의 가격 체계에서 가격 변화 후와 동일한 효용을 얻기 위한 소득 변화다. / 답: O / 설명: 구 가격 체계하에서 측정하는 소득 변화입니다.
17. 가격이 하락할 경우 소비자잉여의 증가분은 보상변동보다 크고 대등변동보다 작다. / 답: X / 설명: 일반적인 정상재의 경우 '보상변동 < 소비자잉여 증가분 < 대등변동'의 관계가 성립합니다.
18. 사회후생함수는 개인들의 효용 수준을 변수로 하여 사회 전체의 후생 수준을 결정하는 함수다. / 답: O / 설명: 사회적 가치판단을 함수 형태로 표현한 것입니다.
19. 벤담의 공리주의 사회후생함수는 사회 구성원들의 효용의 합을 극대화한다. / 답: O / 설명: $W = U_A + U_B$ 형태로 표현되며, 개인별 가중치가 동일합니다.
20. 롤스(Rawls)의 사회후생함수에 따르면 사회에서 가장 가난한 사람의 후생이 증가해야 사회후생이 증가한다. / 답: O / 설명: 최소극대화(maximin) 원칙에 따라 $W = min(U_A, U_B)$입니다.
21. 평등주의적 사회무차별곡선은 원점에 대해 오목한 형태를 띤다. / 답: X / 설명: 평등을 지향할수록 원점에 대해 볼록(convex)한 형태가 되어 극단적 불평등보다 평균적 배분을 선호합니다.
22. 나쉬(Nash)의 사회후생함수는 개인들의 효용을 합산하는 방식이다. / 답: X / 설명: 개인들의 효용을 곱하는 방식($W = U_A \times U_B$)입니다.
23. 가치재(Merit Goods)는 소비자 주권을 존중하기 위해 정부가 개입하지 않는 재화다. / 답: X / 설명: 소비자 주권을 일부 제한하더라도 사회적으로 바람직하다고 판단하여 정부가 소비를 권장하는 재화입니다.
24. 애로우(Arrow)의 불가능성 정리에 따르면 완비성, 이행성, 비독재성 등 합리적 조건을 모두 만족하는 사회적 선호체계는 존재하기 어렵다. / 답: O / 설명: 민주적인 의사결정 방식의 한계를 수학적으로 증명한 것입니다.
25. 지니계수는 0에 가까울수록 소득분배가 평등함을 의미한다. / 답: O / 설명: 0은 완전평등, 1은 완전불평등을 나타냅니다.
26. 로렌츠 곡선이 대각선(45도선)에 가까울수록 소득분배는 불평등해진다. / 답: X / 설명: 45도선은 완전평등선을 의미하므로 가까울수록 평등합니다.
27. 십분위분배율은 값이 클수록 소득분배가 평등함을 나타낸다. / 답: O / 설명: '하위 40% 소득 / 상위 20% 소득'이므로 수치가 클수록 평등합니다.
28. 애킨슨 지수(Atkinson Index)는 사회가 불평등에 대해 얼마나 민감하게 반응하는지를 나타내는 주관적 지수다. / 답: O / 설명: 불평등 혐오도를 반영하여 후생 손실을 측정합니다.
29. 달톤(Dalton)의 비례성 원칙에 따르면 모든 사람의 소득이 동일한 비율로 증가하면 불평등 지수는 변하지 않아야 한다. / 답: O / 설명: 상대적 불평등 지수의 기본 요건 중 하나입니다.
30. 칼도-힉스(Kaldor-Hicks) 기준은 잠재적 보상 가능성만 있다면 실제 보상이 이루어지지 않아도 파레토 개선으로 본다. / 답: O / 설명: 효율성 측면에서의 개선 여부를 판단하는 보상 원칙입니다.
31. 시토프스키(Scitovsky) 기준은 칼도-힉스 기준의 모순(가역성 문제)을 해결하기 위해 제안되었다. / 답: O / 설명: 변화 전후 두 상태에서 모두 칼도-힉스 기준을 만족해야 한다고 봅니다.
32. 보충적 해석에 따르면 사회후생함수는 개인의 선호를 단순히 집계한 것이 아니라 사회적 합의에 의한 가치판단이 개입된 것이다. / 답: O / 설명: 베르그송-사무엘슨 사회후생함수의 특징입니다.
33. 기수적 효용설에 기초한 공리주의는 개인 간 효용 비교가 불가능하다고 가정한다. / 답: X / 설명: 공리주의는 개인 간 효용 비교가 가능하고 합산할 수 있다고 가정합니다.
34. 베르그송-사무엘슨 사회후생함수는 파레토 원칙을 만족한다. / 답: O / 설명: 모든 개인의 효용이 증가하면 사회후생도 반드시 증가한다는 전제를 가집니다.
35. 파레토 효율성은 분배의 형평성에 대한 가치판단을 포함하고 있다. / 답: X / 설명: 자원배분의 효율성만을 다루며 분배가 얼마나 공평한지는 따지지 않습니다.
36. 후생경제학 제1정리가 성립하기 위해서는 시장에 독점이나 외부성이 존재하지 않아야 한다. / 답: O / 설명: 시장실패 요인이 없는 완전경쟁시장을 전제로 합니다.
37. 재분배 정책으로 인한 효율성 상실(사하중 손실)은 후생경제학 제2정리의 실제 적용을 어렵게 만드는 요인이다. / 답: O / 설명: 일시금 조세가 아닌 왜곡적 조세를 사용할 경우 효율성이 저해되기 때문입니다.
38. 생산가능곡선 상의 모든 점은 생산의 효율성을 충족한다. / 답: O / 설명: 주어진 자원으로 생산할 수 있는 최대 조합들의 궤적입니다.
39. 무차별곡선이 L자 형태인 경우(완전보완재), 두 재화의 한계대체율은 정의되지 않는다. / 답: O / 설명: 꺾이는 지점에서 기울기를 구할 수 없기 때문입니다.
40. 사회적 무차별곡선이 수직 또는 수평인 경우, 이는 롤스의 사회후생함수를 나타낸다. / 답: X / 설명: 롤스의 사회무차별곡선은 L자 형태입니다.
41. 효용가능경계(UPF) 상의 한 점은 반드시 파레토 효율적이다. / 답: O / 설명: 파레토 효율적인 배분 상태를 효용 평면으로 옮긴 것입니다.
42. 효용가능곡선은 계약곡선으로부터 유도된다. / 답: O / 설명: 계약곡선의 각 점에서 얻는 효용 조합을 연결한 선입니다.
43. 벤담의 사회후생함수는 소득의 한계효용이 체감할 때 소득의 완전평등 분배를 지지한다. / 답: O / 설명: 한계효용이 일치할 때 총효용 합이 극대화되기 때문입니다.
44. 로버트 노직(Nozik)의 자유지상주의는 결과의 평등보다 과정의 정당성을 중시한다. / 답: O / 설명: 취득과 이전의 과정이 정당하다면 결과가 불평등해도 정의롭다고 봅니다.
45. 지니계수가 1이면 인구의 극소수가 모든 소득을 독점하는 상태다. / 답: O / 설명: 완전불평등 상태를 의미합니다.
46. 앳킨슨 지수는 불평등 혐오 파라미터가 클수록 현재의 불평등을 더 심각하게 평가한다. / 답: O / 설명: 혐오도가 높을수록 동일한 후생을 내는 '균등분배대등소득'이 낮아집니다.

48. 사회선호의 이행성(Transitivity)이란 A보다 B를 선호하고 B보다 C를 선호하면, A보다 C를 선호해야 함을 말한다. / 답: O / 설명: 논리적 일관성을 의미합니다.
49. 콩도르세의 역설(투표의 역설)은 개인의 선호가 이행성을 갖더라도 사회적 선호는 순환할 수 있음을 보여준다. / 답: O / 설명: 다수결 방식의 내재적 결함을 지적합니다.
50. 단봉선호(Single-peaked preference)가 보장되면 다수결 투표에서 이행성이 보장된다. / 답: O / 설명: 투표의 역설이 발생하지 않는 충분조건입니다.
51. 지수적 외부성(Pecuniary Externality)은 상대가격의 변화를 통해 발생하며 자원배분의 효율성을 저해한다. / 답: X / 설명: 단순히 소득 재분배만 일으킬 뿐 기술적 외부성과 달리 효율성을 해치지 않습니다.
52. 보상변동과 대등변동은 소득효과가 0일 때 서로 일치한다. / 답: O / 설명: 소득효과가 없으면 보상수요곡선과 보통수요곡선이 일치하기 때문입니다.

192. 다수결 제도에서 '전략적 투표'란 자신의 진정한 선호와 다르게 투표하여 더 나은 결과를 유도하는 행위다. / 답: O / 설명: 최악을 피하기 위해 차악을 선택하는 행위 등입니다.

194. 합리적 무지로 인해 투표자는 정책의 세부 내용보다 '단순한 구호'에 더 민감하게 반응한다. / 답: O / 설명: 정보 획득 비용을 줄이려는 행동입니다.

196. 관료 조직의 비효율성을 줄이기 위해서는 민간과의 경쟁 체제 도입이 필요하다. / 답: O / 설명: 공공선택이론의 주요 정책 제언입니다.

198. 비용편익분석에서 '사회적 할인율'은 미래 세대의 권리를 보호하기 위한 도덕적 잣대가 될 수 있다. / 답: O / 설명: 낮은 할인율을 통해 미래 가치를 지키려 합니다.

200. 보몰의 병은 공공 서비스의 질을 유지하기 위해서는 지출의 비중이 늘어날 수밖에 없음을 시사한다. / 답: O / 설명: 비용 상승이 구조적이기 때문입니다.

### [제4파트] 조세이론 및 조세의 귀착 OX 문제 200선

1. 조세의 중립성이란 조세가 경제주체의 의사결정과 자원배분에 영향을 주지 않는 상태를 말한다. / 답: O / 설명: 상대가격 체계를 왜곡하지 않는 조세를 의미합니다.
2. 정액세(Lump-sum Tax)는 대체효과가 발생하지 않으므로 중립적인 조세다. / 답: O / 설명: 소득효과만 발생시키고 상대가격을 변화시키지 않아 효율적입니다.
3. 조세의 초과부담(Excess Burden)은 납세자가 납부한 세액 그 자체를 의미한다. / 답: X / 설명: 세금 납부액을 초과하여 발생하는 민간의 후생 손실(사하중 손실)을 의미합니다.
4. 초과부담은 조세로 인해 발생하는 소득효과 때문에 나타난다. / 답: X / 설명: 가격 구조 왜곡에 따른 '대체효과' 때문에 발생합니다.
5. 하버거 삼각형(Harberger Triangle)은 조세의 초과부담을 기하학적으로 표현한 것이다. / 답: O / 설명: 세금으로 인해 줄어든 거래량과 후생 손실을 나타내는 삼각형 면적입니다.
6. 조세의 전가(Shifting)는 법적 납세의무자가 경제적 부담을 타인에게 떠넘기는 과정이다. / 답: O / 설명: 가격 인상 등을 통해 담세자에게 부담을 넘기는 행위입니다.
7. 조세의 귀착(Incidence)은 조세 부담이 최종적으로 누구에게 머무르는가를 의미한다. / 답: O / 설명: 법적 귀착과 경제적 귀착은 다를 수 있습니다.
8. 수요가 탄력적일수록 조세 부담은 소비자에게 더 많이 귀착된다. / 답: X / 설명: 탄력적인 쪽(선택지가 많은 쪽)이 세금을 적게 부담합니다.
9. 공급이 완전비탄력적이면 조세는 전액 공급자에게 귀착된다. / 답: O / 설명: 공급자가 생산량을 조절할 수 없으므로 세금을 다 떠안게 됩니다.
10. 수요와 공급의 탄력성이 같으면 조세는 생산자와 소비자가 절반씩 부담한다. / 답: O / 설명: 부담 비율은 탄력성의 역비율에 비례합니다.
11. 조세 부과 시 법적 납세의무자가 누구냐에 따라 최종적인 경제적 귀착 결과가 달라진다. / 답: X / 설명: 시장의 힘(탄력성)에 의해 결정되므로 법적 의무자와는 무관합니다.
12. 비례세는 모든 소득 수준에 대해 동일한 세율을 적용하는 세금이다. / 답: O / 설명: 세액은 소득에 비례하여 늘어나지만 세율은 일정합니다.
13. 누진세는 소득이 증가할수록 한계세율이 높아지는 구조를 가진다. / 답: O / 설명: 수직적 형평성을 달성하기 위한 대표적 수단입니다.
14. 역진세는 소득이 높은 사람일수록 소득 대비 세액 비중이 낮아지는 세금이다. / 답: O / 설명: 저소득층의 부담이 상대적으로 큰 세금입니다.
15. 편익 원칙(Benefit Principle)은 공공 서비스로부터 얻는 편익에 따라 과세해야 한다는 원칙이다. / 답: O / 설명: 수익자 부담 원칙과 일맥상통하며 효율적 자원배분을 중시합니다.
16. 능력 원칙(Ability-to-pay Principle)은 개인의 경제적 지불 능력에 따라 세금을 부과하는 원칙이다. / 답: O / 설명: 소득 재분배와 형평성을 강조하는 원칙입니다.
17. 수평적 형평성은 동일한 경제적 능력을 가진 사람은 동일한 세금을 내야 한다는 것이다. / 답: O / 설명: 동일 처우의 원칙입니다.
18. 수직적 형평성은 능력이 큰 사람이 더 많은 세금을 내야 한다는 원칙이다. / 답: O / 설명: 차등 처우를 통한 공평성 달성입니다.
19. 램지(Ramsey) 규칙에 따르면 초과부담을 최소화하기 위해 수요의 탄력성이 높은 재화에 높은 세율을 적용해야 한다. / 답: X / 설명: 역탄력성 원칙에 따라 탄력성이 '낮은' 재화에 높은 세율을 적용해야 합니다.
20. 콜렛-헤이그(Corlett-Hague) 규칙은 여가와 보완적인 재화에 높은 세율을 부과할 것을 권고한다. / 답: O / 설명: 직접 과세가 불가능한 여가의 소비를 간접적으로 억제하여 효율성을 높이려는 전략입니다.
21. 직접세는 납세의무자와 담세자가 일치할 것을 예정하는 조세다. / 답: O / 설명: 소득세, 법인세 등이 해당합니다.
22. 간접세는 세금의 전가를 전제로 하여 납세의무자와 담세자가 다른 조세다. / 답: O / 설명: 부가가치세, 개별소비세 등이 해당합니다.
23. 조세의 초과부담은 세율의 제곱에 비례하여 증가한다. / 답: O / 설명: 세율이 조금만 올라도 초과부담은 급격히(기하급수적으로) 커집니다.
24. 종량세는 물품의 수량을 기준으로, 종가세는 물품의 가격을 기준으로 과세하는 방식이다. / 답: O / 설명: 과세 표준의 측정 단위에 따른 구분입니다.
25. 래퍼 곡선(Laffer Curve)은 세율이 지나치게 높으면 노동 의욕 저하로 오히려 세수가 감소할 수 있음을 보여준다. / 답: O / 설명: 최적 세수를 실현하는 세율이 존재한다는 이론입니다.
26. 조세의 자산화(Capitalization)는 미래의 세금 부담이 자산의 현재 가격에 반영되어 하락하는 현상이다. / 답: O / 설명: 토지세 도입 시 토지 가격이 세액의 현재가치만큼 떨어지는 것이 예입니다.
27. 일반불균형 분석은 특정 시장뿐만 아니라 경제 전체의 연관 관계를 고려하여 조세의 귀착을 분석한다. / 답: O / 설명: 하버거(Harberger) 모형이 대표적입니다.
28. 부분불균형 분석은 다른 시장의 변화는 무시하고 해당 시장 내의 공급과 수요만으로 귀착을 분석한다. / 답: O / 설명: 마샬(Marshall)적 분석 방식입니다.
29. 소득세의 과세 베이스를 '포괄적 소득'으로 잡아야 한다는 주장은 헤이그-시몬스 정의에 기초한다. / 답: O / 설명: 소비와 순자산 가치의 증가분을 모두 소득으로 봅니다.
30. 세액 공제는 소득 공제보다 고소득자에게 더 유리한 방식이다. / 답: X / 설명: 세액 공제는 동일 금액을 깎아주므로 저소득자에게 상대적으로 더 유리하거나 공평합니다.
31. 법인세의 이중과세 문제는 법인 단계에서 과세된 이익이 주주 배당 시 소득세로 다시 과세되는 것을 말한다. / 답: O / 설명: 이를 해결하기 위해 배당소득 세액공제(Imputation) 시스템을 사용합니다.
32. 부가가치세는 전단계 세액공제법을 통해 계산된다. / 답: O / 설명: 매출세액에서 매입세액을 차감하여 납부세액을 정합니다.
33. 소비지국 과세 원칙은 수출품에 대해서는 영세율을 적용하여 이중과세를 방지하는 원칙이다. / 답: O / 설명: 국제 거래에서 부가가치세의 혼란을 막기 위한 약속입니다.
34. 면세(Exemption)는 부가가치세 부담을 완전히 제거하는 제도다. / 답: X / 설명: 면세는 매출세액만 면제할 뿐 매입세액은 환급되지 않으므로 부분적 면세입니다. 영세율이 완전 면세입니다.
35. 조세의 중립적 배분을 위해 모든 재화에 동일한 세율을 적용해야 한다는 것이 램지 규칙의 결론이다. / 답: X / 설명: 탄력성에 따라 세율을 다르게 적용해야(역탄력성) 효율적이라고 봅니다.
36. 대체효과가 0인 재화에 과세하면 초과부담은 0이 된다. / 답: O / 설명: 왜곡이 발생하지 않기 때문입니다.
37. 수입 관세 부과 시 국내 소비자 가격은 관세액만큼 반드시 상승한다. / 답: X / 설명: 국내외 수급 탄력성에 따라 상승 폭은 달라질 수 있습니다.
38. 법인세는 단기적으로 전가가 어렵지만 장기적으로 자본 이동을 통해 소비자나 노동자에게 전가될 수 있다. / 답: O / 설명: 자본이 다른 산업으로 빠져나가면 공급이 줄어 가격이 오르거나 임금이 내릴 수 있습니다.
39. 소득세가 누진적일수록 자동안정화 장치로서의 기능이 강화된다. / 답: O / 설명: 경기 과열 시 세수가 더 빠르게 늘어 경기를 진정시키기 때문입니다.
40. 조세의 '정태적 귀착'은 조세 부과 직후의 가격 변화를, '동태적 귀착'은 자원 이동 후의 장기적 변화를 분석한다. / 답: O / 설명: 시간적 범위에 따른 구분입니다.
41. 조세 부과 시 소비자 가격이 100원 오르고 생산자 수취 가격이 50원 내렸다면, 조세 총액은 150원이다. / 답: O / 설명: 소비자 가격과 생산자 가격의 차이(Tax Wedge)가 세금입니다.
42. 독점 시장에서 종량세 부과 시 가격 상승 폭은 세금 액수보다 클 수도 있다. / 답: O / 설명: 한계비용 곡선의 기울기와 수요곡선의 형태에 따라 가능합니다.
43. 후방 전가(Backward Shifting)는 세금 부담을 생산 요소(노동, 자본) 공급자에게 떠넘기는 것이다. / 답: O / 설명: 임금을 깎거나 원재료 값을 깎는 행위입니다.
44. 최적 소득세 이론에 따르면 한계세율을 지나치게 높이면 노동 공급의 왜곡이 커져 사회후생이 감소한다. / 답: O / 설명: 미르리스(Mirrlees) 등의 연구 결과입니다.
45. 조세의 중립성은 항상 사회적 정의(형평성)와 일치한다. / 답: X / 설명: 중립적인 정액세는 효율적이지만 저소득층에게 가혹하여 불평등을 심화시킬 수 있습니다.
46. 소득세법상 '결집 효과'는 여러 해에 걸쳐 발생한 소득이 한꺼번에 실현될 때 누진세율로 인해 과도한 세금이 부과되는 현상이다. / 답: O / 설명: 퇴직소득이나 양도소득에서 주로 발생합니다.
47. 물가 상승으로 인해 실질 소득은 그대로인데 명목 소득 상승으로 높은 세율 구간에 진입하는 것을 '브래킷 크리프'라고 한다. / 답: O / 설명: 인플레이션에 따른 자동 증세 효과입니다.
