
### [부가가치세법] OX 문제 100선 (001~100)

1. 부가가치세의 납세의무자는 사업상 독립적으로 재화 또는 용역을 공급하는 자이다. / 답: O / 설명: 사업성(계속·반복성)과 독립성이 납세의무자의 핵심 요건입니다.
2. 영리 목적이 없는 비영리법인은 부가가치세 납세의무를 지지 않는다. / 답: X / 설명: 부가가치세는 영리 목적 유무를 불문하고 사업자 요건을 충족하면 납세의무가 있습니다.
3. 국가 및 지방자치단체도 부가가치세법상 납세의무자가 될 수 있다. / 답: O / 설명: 국가 등도 과세 대상 거래를 하면 납세의무를 집니다.
4. 부가가치세는 납세의무자와 담세자가 다른 간접세에 해당한다. / 답: O / 설명: 사업자가 세금을 내지만, 실질적인 부담은 최종소비자가 합니다.
5. 우리나라는 생산지국 과세원칙을 채택하여 수출 시 과세한다. / 답: X / 설명: 소비지국 과세원칙을 채택하여 수출 시 영세율을 적용합니다.
6. 신규로 사업을 시작하려는 자는 사업 개시일 전이라도 사업자등록을 신청할 수 있다. / 답: O / 설명: 매입세액 공제 등을 위해 개업 전 등록을 허용합니다.
7. 사업자등록 신청을 받은 세무서장은 신청일부터 2일 이내에 등록증을 발급해야 한다. / 답: O / 설명: 원칙적인 발급 기한입니다(조사가 필요한 경우 제외).
8. 주사업장 총괄납부 사업자는 각 사업장별로 부가가치세를 신고해야 한다. / 답: O / 설명: 총괄납부는 '납부'만 통합할 뿐, '신고'는 사업장별로 각각 해야 합니다.
9. 사업자단위과세 사업자는 본점 또는 주사무소에서 일괄하여 신고와 납부를 모두 수행한다. / 답: O / 설명: 총괄납부보다 더 통합된 형태의 제도입니다.
10. 재화의 공급이란 재화의 소유권을 이전하는 모든 유상·무상의 거래를 말한다. / 답: X / 설명: 무상 공급은 원칙적으로 공급이 아니며, 예외적인 '간주공급'만 과세됩니다.
11. 수표, 어음 등 화폐대용증권의 양도는 부가가치세 과세대상이다. / 답: X / 설명: 화폐대용증권은 부가가치세법상 '재화'에 해당하지 않습니다.
12. 담보의 제공은 재화의 실질적 공급으로 보지 않는다. / 답: O / 설명: 소유권의 실질적 이전이 없으므로 과세하지 않습니다.
13. 사업장별로 그 사업에 관한 모든 권리와 의무를 포괄적으로 승계시키는 사업양도는 재화의 공급으로 보지 않는다. / 답: O / 설명: 불필요한 자금 압박 방지를 위한 비과세 규정입니다.
14. 조세의 물납은 재화의 공급으로 보아 부가가치세를 과세한다. / 답: X / 설명: 강제적 납부 성격이므로 공급으로 보지 않습니다.
15. 자가공급 중 면세사업 전용 재화는 매입세액이 공제된 것에 한하여 과세한다. / 답: O / 설명: 간주공급은 공제받은 세액을 추징하는 성격입니다.
16. 비영업용 소형승용차의 구입 및 유지는 매입세액 공제 여부와 상관없이 무조건 간주공급으로 과세한다. / 답: X / 설명: 매입세액이 공제된 경우에만 간주공급(자가공급)으로 과세합니다.
17. 사업상 증여는 증여하는 재화의 매입세액이 공제되지 않은 경우에도 공급으로 본다. / 답: X / 설명: 매입세액이 공제되지 않았다면 공급으로 의제하지 않습니다.
18. 폐업 시 잔존재산은 폐업일에 자기에게 공급한 것으로 보아 과세한다. / 답: O / 설명: 소비되지 않은 재고에 대해 마지막으로 과세하는 절차입니다.
19. 용역의 무상 공급은 원칙적으로 과세대상에서 제외된다. / 답: O / 설명: 용역은 재화와 달리 무상 거래를 포착하기 어려워 과세하지 않습니다.
20. 특수관계인에게 사업용 부동산 임대용역을 무상으로 제공하는 것은 과세대상이다. / 답: O / 설명: 용역의 무상 공급 중 유일한 과세 예외 규정입니다.
21. 고용관계에 따라 근로를 제공하는 것은 부가가치세법상 용역의 공급이다. / 답: X / 설명: 독립성이 없으므로 과세 대상이 아닙니다.
22. 재화의 공급시기는 원칙적으로 재화가 인도되는 때이다. / 답: O / 설명: 이동이 필요한 재화의 일반적인 원칙입니다.
23. 장기할부판매의 공급시기는 대금의 각 부분을 받기로 한 때이다. / 답: O / 설명: 대금 회수 기간이 길어 납세자의 부담을 고려한 특례입니다.
24. 무인판매기를 통한 재화 공급은 현금을 인출하는 때를 공급시기로 본다. / 답: O / 설명: 거래 시점을 일일이 확인하기 어려운 업종의 특징입니다.
25. 수출 재화의 공급시기는 배에 물건을 싣는 날(선적일)이다. / 답: O / 설명: 수출의 과세 시점 판정 기준입니다.
26. 용역의 공급시기는 원칙적으로 역무의 제공이 완료되는 때이다. / 답: O / 설명: 서비스가 다 제공된 시점이 기준입니다.
27. 부동산 임대용역 중 간주임대료의 공급시기는 예정신고기간 또는 과세기간의 종료일이다. / 답: O / 설명: 실제 대가가 없으므로 기간 종료일을 기준으로 계산합니다.
28. 영세율은 매출세액은 0이고 매입세액은 전액 환급되는 '완전면세' 제도이다. / 답: O / 설명: 부가가치세 부담을 완전히 제거합니다.
29. 면세는 매출세액은 없으나 매입 시 부담한 세액은 환급되지 않는 '부분면세' 제도이다. / 답: O / 설명: 최종소비자의 세부담만 경감시키는 효과가 있습니다.
30. 면세사업자가 영세율을 적용받으려면 반드시 면세포기 신고를 해야 한다. / 답: O / 설명: 면세와 영세율은 동시에 적용될 수 없습니다.
31. 식용으로 제공되는 국산 미가공 식료품은 면세이나 수입산은 과세이다. / 답: X / 설명: 식용 미가공 식료품은 국산·수입 따지지 않고 모두 면세입니다.
32. 비식용 미가공 축산물은 국산인 경우에만 면세된다. / 답: O / 설명: 비식용은 국내 생산 장려를 위해 국산만 면세합니다.
33. 수돗물은 면세이나 전기는 과세이다. / 답: O / 설명: 기초 생활필수품 여부의 차이입니다.
34. 연탄과 무연탄은 면세이지만 유연탄은 과세 대상이다. / 답: O / 설명: 서민 연료 보호 차원입니다.
35. 시내버스, 지하철은 면세이나 택시, KTX, 항공기는 과세이다. / 답: O / 설명: 대중교통 수단 중심의 면세 범위입니다.
36. 도서, 신문, 잡지는 면세이나 그 광고료는 과세된다. / 답: O / 설명: 정보 전달 매체 자체는 면세하되 상업적 광고는 과세합니다.
37. 의료보건 용역 중 성형수술, 애완동물 진료 등은 과세 대상이다. / 답: O / 설명: 필수 의료 서비스가 아니라고 판단하기 때문입니다.
38. 은행업무 중 금전 대여에 따른 이자 수입은 면세 대상이다. / 답: O / 설명: 금융 용역의 가치 포착이 어려워 면세로 규정합니다.
39. 토지의 양도는 면세이나 토지의 임대는 과세이다. / 답: O / 설명: 자본의 이전은 면세, 서비스의 제공은 과세입니다.
40. 주택의 임대와 그 부수토지의 임대 용역은 면세이다. / 답: O / 설명: 서민 주거 안정을 위한 면세 규정입니다.
41. 우체국 택배와 국가가 운영하는 숙박업은 과세 대상이다. / 답: O / 설명: 민간 업체와의 형평성을 위해 과세로 전환되었습니다.

44. 매출에누리와 매출환입은 과세표준에서 직접 공제한다. / 답: O / 설명: 실질적인 공급 가액이 줄어들었기 때문입니다.
45. 공급받는 자에게 도달하기 전에 파손된 재화의 가액은 과세표준에 포함하지 않는다. / 답: O / 설명: 공급이 완료되지 않은 것으로 봅니다.
46. 대손금은 과세표준에서 직접 공제한다. / 답: X / 설명: 대손금은 과세표준에서 빼는 것이 아니라 '매출세액'에서 공제합니다.
