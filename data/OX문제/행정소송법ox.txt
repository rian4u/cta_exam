### 행정소송법 OX 문제 100선

1. 행정소송은 주관적 소송인 항고소송·당사자소송과 객관적 소송인 민중소송·기관소송으로 구분된다. / 답: O / 설명: 행정소송법 제3조의 소송 종류 구분입니다.
2. 취소소송은 행정청의 처분등을 취소하거나 변경하는 소송이다. / 답: O / 설명: 대표적인 항고소송의 종류입니다.
3. 부작위위법확인소송은 행정청의 부작위가 위법함을 확인하는 소송으로 항고소송에 해당한다. / 답: O / 설명: 행정소송법 제4조 제3호에 명시되어 있습니다.
4. 민중소송과 기관소송은 법률이 정한 경우에 법률이 정한 자에 한하여 제기할 수 있는 법정소송이다. / 답: O / 설명: 객관적 소송의 특성상 법률의 명문 규정이 필수적입니다.
5. 행정소송에 관하여 행정소송법에 특별한 규정이 없는 사항은 민사소송법과 민사집행법의 규정을 준용한다. / 답: O / 설명: 행정소송법 제8조 제2항에 따른 준용 규정입니다.
6. 취소소송의 제1심 관할법원은 피고의 소재지를 관할하는 행정법원이다. / 답: O / 설명: 소재지주의 관할 원칙입니다.
7. 중앙행정기관의 장이 피고인 경우 대법원 소재지의 행정법원(서울행정법원)에도 소를 제기할 수 있다. / 답: O / 설명: 관할의 특례 규정입니다.
8. 토지수용 등 부동산에 관한 처분의 경우 그 부동산 소재지의 관할 행정법원에 소를 제기할 수 있다. / 답: O / 설명: 부동산 소재지 관할의 인정입니다.
9. 관련청구소송은 본래의 취소소송이 계속된 법원에 병합하여 제기할 수 있다. / 답: O / 설명: 소송의 효율성을 위한 관련청구의 병합 제도입니다.
10. 관련청구소송의 병합은 사실심의 변론종결 시까지만 가능하다. / 답: O / 설명: 상고심에서의 병합은 허용되지 않습니다.
11. 취소소송의 피고는 원칙적으로 해당 처분을 행한 행정청이다. / 답: O / 설명: 행정청이 피고가 되는 항고소송의 특징입니다.
12. 대통령의 처분의 경우 소속 장관을 피고로 하여야 한다. / 답: O / 설명: 행정소송법 제13조 단서에 따른 예외입니다.
13. 피고를 잘못 지정한 경우 법원은 원고의 신청에 의하여 결정으로써 피고를 경정하게 하여야 한다. / 답: O / 설명: 피고경정 제도입니다.
14. 피고경정의 결정이 있는 경우 새로운 피고에 대한 소송은 처음 소를 제기한 때에 제기된 것으로 본다. / 답: O / 설명: 제소기간 준수를 위한 소급효 인정입니다.
15. 제3자에 의한 소송참가는 법원의 결정으로써 허용되며, 당사자의 신청 또는 직권으로 가능하다. / 답: O / 설명: 제3자의 권리 보호를 위한 소송참가 규정입니다.
16. 행정청이 소송에 참가하는 경우 법원은 당사자나 해당 행정청의 신청에 의해서만 결정할 수 있다. / 답: X / 설명: 당사자의 신청 또는 직권으로 결정할 수 있습니다.
17. 취소소송은 처분등이 있음을 안 날부터 90일 이내에 제기하여야 한다. / 답: O / 설명: 주관적 제소기간입니다.
18. 처분등이 있은 날부터 1년이 지나면 정당한 사유가 없는 한 소를 제기하지 못한다. / 답: O / 설명: 객관적 제소기간입니다.
19. 행정심판을 거친 경우의 제소기간은 재결서의 정본을 송달받은 날부터 기산한다. / 답: O / 설명: 행정심판 전치 시 제소기간의 특례입니다.
20. 제소기간의 90일은 불변기간이다. / 답: O / 설명: 행정소송법 제20조 제3항에 명시되어 있습니다.
21. 행정심판은 원칙적으로 필수적 전심절차이다. / 답: X / 설명: 원칙은 임의적 전치주의이며, 법률에 규정이 있는 경우에만 필수적입니다.
22. 국세처분이나 지방세처분을 다투는 소송은 반드시 행정심판을 거쳐야 한다. / 답: O / 설명: 관련 개별법에 따른 필수적 전치의 예입니다.
23. 필요적 전치주의가 적용되는 경우에도 행정심판의 의결이 없으면 60일이 지난 후 재결 없이 소를 제기할 수 있다. / 답: O / 설명: 재결을 기다리지 않고 소를 제기할 수 있는 예외 사유입니다.
24. 처분의 집행으로 생길 중대한 손해를 예방하여야 할 긴급한 필요가 있는 경우 재결 없이 소를 제기할 수 있다. / 답: O / 설명: 재결을 거치지 않아도 되는 사유 중 하나입니다.
25. 취소소송의 제기는 처분의 효력이나 집행 또는 절차의 속행에 영향을 주지 아니한다. / 답: O / 설명: 집행부정지의 원칙입니다.
26. 집행정지는 처분등으로 인하여 생길 회복하기 어려운 손해를 예방하기 위한 제도이다. / 답: O / 설명: 집행정지의 목적입니다.
27. 집행정지 결정은 법원의 직권으로도 할 수 있다. / 답: O / 설명: 신청 또는 직권으로 결정 가능합니다.
28. 집행정지는 본안소송이 법원에 계속 중이어야 신청할 수 있다. / 답: O / 설명: 본안의 계속이 집행정지의 전제 조건입니다.
29. 공공복리에 중대한 영향을 미칠 우려가 있는 때에는 집행정지가 허용되지 아니한다. / 답: O / 설명: 집행정지의 소극적 요건입니다.
30. 집행정지 결정에 대하여는 즉시항고할 수 있으며, 이 경우 결정의 집행이 정지되는 효력이 있다. / 답: X / 설명: 즉시항고는 가능하나 결정의 집행을 정지하는 효력은 없습니다.
31. 취소소송에서 법원은 필요하다고 인정할 때에는 당사자가 주장하지 아니한 사실에 대하여도 심리할 수 있다. / 답: O / 설명: 직권심리주의의 인정(제26조)입니다.
32. 법원은 행정심판기록의 제출을 명할 수 있으며, 이는 당사자의 신청이 있는 경우에만 가능하다. / 답: X / 설명: 당사자의 신청 또는 직권으로 제출 명령을 할 수 있습니다.
33. 소의 종류의 변경은 피고가 동의하는 경우에만 허용된다. / 답: X / 설명: 법원은 원고의 신청에 의하여 결정으로써 소의 종류를 변경할 수 있습니다.
34. 처분변경으로 인한 소의 변경은 처분의 변경이 있음을 안 날부터 60일 이내에 신청하여야 한다. / 답: O / 설명: 소 변경의 신청 기간 제한입니다.
35. 행정소송의 판결은 당사자 외에 제3자에게도 효력이 있다. / 답: O / 설명: 형성판결의 대세효입니다.
36. 처분을 취소하는 판결이 확정된 경우 그 사건에 대하여 피고인 행정청을 기속한다. / 답: O / 설명: 판결의 기속력입니다.
37. 기속력은 판결의 주문뿐만 아니라 그 전제가 된 구체적 위법사유에 관한 판단에도 미친다. / 답: O / 설명: 기속력의 범위에 관한 판례의 입장입니다.
38. 취소판결이 확정된 후 행정청이 동일한 사유로 다시 처분을 하는 것은 기속력에 위반되어 당연무효이다. / 답: O / 설명: 반복금지의무 위반의 효과입니다.
39. 거부처분 취소판결이 확정된 경우 행정청은 판결의 취지에 따라 다시 처분을 하여야 한다. / 답: O / 설명: 재처분의무의 발생입니다.
40. 간접강제는 거부처분 취소소송이나 부작위위법확인소송에서 재처분의무를 이행하지 않을 때 사용된다. / 답: O / 설명: 재처분의무 실효성 확보 수단입니다.
41. 사정판결은 원고의 청구가 이유 있음에도 처분을 취소하는 것이 공공복리에 적합하지 않다고 인정할 때 내리는 기각판결이다. / 답: O / 설명: 행정소송법 제28조 사정판결의 정의입니다.
42. 사정판결을 하는 경우 원고는 피고인 행정청이 속하는 국가 또는 공공단체를 상대로 손해배상을 청구할 수 있다. / 답: O / 설명: 사정판결에 따른 구제책입니다.
43. 무효등확인소송에는 사정판결 규정이 준용되지 않는다. / 답: O / 설명: 무효인 처분은 처음부터 효력이 없으므로 사정판결을 할 수 없습니다.
44. 무효등확인소송은 제소기간의 제한을 받지 않는다. / 답: O / 설명: 언제든지 제기할 수 있는 것이 원칙입니다.

46. 부작위위법확인소송의 원고적격은 부작위의 확인을 구할 법률상 이익이 있는 자에게 인정된다. / 답: O / 설명: 처분을 신청한 자로서 권리를 침해당한 자입니다.
47. 부작위위법확인소송에서 '부작위'란 행정청이 신청에 대하여 일정한 처분을 하여야 할 법률상 의무가 있음에도 하지 않는 것을 말한다. / 답: O / 설명: 부작위의 정의입니다.
48. 당사자소송의 피고는 국가·공공단체 그 밖의 권리주체이다. / 답: O / 설명: 행정청이 아닌 법인격 주체가 피고가 됩니다.
49. 공법상 계약에 관한 분쟁은 당사자소송으로 다루어야 한다. / 답: O / 설명: 판례에 따른 당사자소송의 예입니다.
