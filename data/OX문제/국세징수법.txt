### [제1파트] 국세징수법 OX 문제 100선 (총칙 ~ 임의징수)

1. 국세징수법은 국세의 징수에 필요한 사항을 규정하여 징수행정의 적정한 운영을 목적으로 한다. / 답: O / 설명: 국세징수법 제1조 목적 규정입니다.
2. 국세징수법은 국세기본법 및 다른 세법에 우선하여 적용된다. / 답: X / 설명: 국세기본법이나 개별 세법에 특별한 규정이 있는 경우를 제외하고는 국세징수법을 적용합니다.
3. '체납자'란 국세를 납부기한까지 납부하지 아니한 자를 말한다. / 답: O / 설명: 국세징수법상 체납자의 정의입니다.
4. '강제징수'란 납세자가 체납한 경우 국가가 강제적으로 자산을 압류·매각하는 절차를 말한다. / 답: O / 설명: 종전의 '체납처분'이 '강제징수'로 용어가 변경되었습니다.
5. 세무서장은 국세를 징수하려면 납세자에게 납부할 국세의 종류, 세액, 기한, 장소를 적은 납부고지서를 발송해야 한다. / 답: O / 설명: 납부 고지의 기본 원칙입니다.
6. 납부고지서에는 가산세의 산출근거를 생략할 수 있다. / 답: X / 설명: 가산세가 포함된 경우 그 종류와 산출근거를 반드시 명시해야 합니다.
7. 국세의 납부기한은 고지서를 발송한 날부터 15일 이내로 정하는 것이 원칙이다. / 답: X / 설명: 납부고지서를 발송하는 경우 납부기한은 발송일부터 14일 이후로 정합니다.
8. 납세자가 도달한 날부터 14일 이내에 국세를 납부하지 않으면 즉시 강제징수에 착수한다. / 답: X / 설명: 독촉 절차를 먼저 거쳐야 합니다.
9. 국세징수법상 '납부기한'은 세법에 따른 법정신고기한과 동일한 의미이다. / 답: X / 설명: 법정납부기한(신고기한)과 지정납부기한(고지서상 기한)은 구분됩니다.
10. 고액·상습체납자에 대한 감치 제도는 국세징수법에 규정되어 있다. / 답: O / 설명: 법원의 결정으로 체납자를 유치장에 가두는 제도입니다.
11. 세무서장은 납세자가 국세를 완납할 수 없다고 인정되는 경우 납부기한을 연장할 수 있다. / 답: O / 설명: 재난, 사업상 위기 등의 사유 시 납부기한 연장이 가능합니다.
12. 납부기한 연장 기간은 9개월을 초과할 수 없는 것이 원칙이다. / 답: O / 설명: 국세징수법상 납부기한 연장 한도입니다.
13. 납부기한 연장 시 관할 세무서장은 납세담보의 제공을 요구할 수 없다. / 답: X / 설명: 기한 연장 시 필요하다고 인정되면 담보 제공을 명할 수 있습니다.
14. 고지된 국세의 납부기한을 다시 연장하는 것을 '징수유예'라고 한다. / 답: O / 설명: 현재는 '납부기한등의 연장'으로 용어가 통합 관리되기도 합니다.
15. 납기전징수 사유가 발생한 경우 납부기한 전이라도 국세를 징수할 수 있다. / 답: O / 설명: 조세채권의 조기 확보를 위한 제도입니다.
16. 납세자가 국외로 이주하기 위해 출국하는 경우는 납기전징수 사유에 해당한다. / 답: O / 설명: 국세징수법 제9조 사유 중 하나입니다.
17. 납기전징수를 하려면 반드시 새로운 납부고지서를 발송해야 한다. / 답: O / 설명: 기한을 앞당겨 고지해야 합니다.
18. 독촉장은 납부기한이 지난 후 10일 이내에 발송해야 한다. / 답: O / 설명: 독촉장 발송 시한 규정입니다.
19. 독촉장에 의한 납부기한은 발송일부터 20일 이내로 정한다. / 답: O / 설명: 독촉 시 부여하는 추가 납부 기간입니다.
20. 독촉은 강제징수(압류)를 하기 위한 필수적인 전제 조건이다. / 답: O / 설명: 독촉 없이 압류하는 것은 원칙적으로 위법입니다 (단, 납기전징수 시 예외).
21. 국세징수법상 제3자 채무자에 대한 채권 압류 통지는 서면으로 해야 한다. / 답: O / 설명: 채권 압류의 효력 발생 요건입니다.
22. 체납자가 사망한 경우 강제징수 절차는 중단되며 상속인에게 새로 고지해야 한다. / 답: X / 설명: 이미 착수한 강제징수는 체납자가 사망하더라도 재산에 대하여 계속 진행할 수 있습니다.
23. 세무서장은 체납자의 재산을 압류하기 위해 필요한 경우 가옥이나 창고를 수색할 수 있다. / 답: O / 설명: 수색권에 관한 규정입니다.
24. 수색은 해가 뜨기 전이나 해가 진 후에는 할 수 없는 것이 원칙이다. / 답: O / 설명: 야간 수색의 제한 원칙입니다 (단, 영업 중인 장소 등 예외 있음).
25. 압류한 동산이나 유가증권은 세무공무원이 직접 보관해야 하며 제3자에게 맡길 수 없다. / 답: X / 설명: 운반이 곤란하거나 부적당한 경우 제3자나 체납자에게 보관시킬 수 있습니다.
26. 체납자의 생활에 없어서는 안 될 식복, 가구 등은 압류할 수 없다. / 답: O / 설명: 절대적 압류금지 재산입니다.
27. 훈장이나 기장은 압류할 수 있는 재산에 해당한다. / 답: X / 설명: 훈장, 기장 등은 압류가 금지됩니다.
28. 급여채권은 그 전액을 압류할 수 있다. / 답: X / 설명: 생활 보장을 위해 일정 금액(보통 1/2)은 압류가 제한됩니다.
29. 퇴직금 등은 그 금액의 2분의 1에 해당하는 금액을 압류하지 못한다. / 답: O / 설명: 압류금지 채권의 범위입니다.
30. 주택임대차보호법에 따른 소액보증금 중 최우선변제금은 압류할 수 없다. / 답: O / 설명: 최저생계비 보장 차원입니다.
31. 압류의 효력은 압류 재산으로부터 발생하는 천연과실에는 미치지 않는다. / 답: X / 설명: 압류의 효력은 종물 및 과실에도 미칩니다.
32. 세무서장은 압류한 재산이 부패할 우려가 있는 경우 즉시 매각할 수 있다. / 답: O / 설명: 긴급 매각 사유입니다.
33. 압류 후 국세 부과의 취소 결정이 있으면 압류를 즉시 해제해야 한다. / 답: O / 설명: 압류 해제의 필수 사유입니다.
34. 압류 재산의 가액이 징수비와 국세에 미달함이 명백한 경우 압류를 해제할 수 있다. / 답: O / 설명: 무익한 압류의 해제 규정입니다.
35. 제3자가 압류 재산에 대하여 소유권을 주장하며 증명하는 경우 압류를 해제해야 한다. / 답: O / 설명: 제3자 소유권 주장 시의 처리입니다.

37. 공매 공고는 매각 결정일 10일 전까지 해야 한다. / 답: O / 설명: 공매 공고 기간 규정입니다.
38. 체납자는 본인의 재산 공매에 직접 입찰하여 매수할 수 있다. / 답: X / 설명: 체납자는 매수인이 될 수 없습니다.
39. 세무공무원은 공매 재산을 직접 매수할 수 없다. / 답: O / 설명: 직무 관련자의 매수 금지 규정입니다.
40. 공매 낙찰자가 대금을 기한 내에 납부하지 않으면 매각 결정을 취소한다. / 답: O / 설명: 대금 미납 시의 조치입니다.

42. 배분액에 이의가 있는 채권자는 배분기일부터 30일 이내에 소송을 제기해야 한다. / 답: X / 설명: 배분 계산서에 대한 이의는 배분기일에 즉시 제기해야 합니다.
