# OX문제

공용 OX 텍스트 원본 폴더입니다.

## 역할

- 과목별 OX 문제 텍스트를 저장
- [scripts/import_ox_text.py](/e:/Project/tax_exam3/scripts/import_ox_text.py)가 이 폴더를 읽어 `OX` 테이블에 적재
- OX 모드와 게임 모드는 최종적으로 DB의 `OX` 테이블을 조회

## 현재 파일 규칙

- 과목명 기반 파일명 사용
  - 예: `재정학ox.txt`, `상법ox.txt`, `국세기본법.txt`
- 대소문자 차이는 스크립트에서 완화 처리
- 연도 접두가 필요하면 `2025_파일명` 형태도 지원

## 입력 데이터 형식

스크립트는 기본적으로 아래 3요소를 파싱합니다.

- 문제
- 답(O/X)
- 설명(해설)

문제번호가 중복되어도, 현재 적재 로직은 파일 등장 순서대로 모두 적재합니다.

## 운영 메모

- 세법학개론은 세목별 파일로 세분화되어 있음
- 이 폴더를 수정한 뒤에는 재적재가 필요함
