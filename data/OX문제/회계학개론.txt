### [PART 4. 원가관리회계 핵심 이론 (251~400)]

251. 원가회계는 내부 의사결정자뿐만 아니라 외부 정보이용자에게도 유용한 정보를 제공한다. / 답: O / 설명: 재무제표의 매출원가 및 재고자산 가액 결정에 기여하기 때문입니다.
252. 직접원가는 특정 원가대상에 추적이 가능한 원가를 의미하며, 간접원가는 배부 절차가 필요하다. / 답: O / 설명: 추적 가능성에 따른 원가 분류입니다.
253. 변동원가는 조업도가 증가함에 따라 단위당 원가가 일정하게 유지된다. / 답: O / 설명: 총원가는 증가하지만 단위당 원가는 불변입니다.
254. 고정원가는 조업도가 증가함에 따라 총원가는 일정하지만 단위당 원가는 감소한다. / 답: O / 설명: 고정비의 특성입니다.
255. 기회원가는 여러 대안 중 하나를 선택함에 따라 포기한 대안들 중 가치가 가장 큰 것을 의미한다. / 답: O / 설명: 의사결정 시 고려해야 할 경제적 원가입니다.
256. 매몰원가는 과거에 이미 발생한 원가로서 현재나 미래의 의사결정에 고려해서는 안 된다. / 답: O / 설명: 의사결정의 비관련원가입니다.
257. 기초원가(기본원가)는 직접재료원가와 직접노무원가의 합이다. / 답: O / 설명: 생산의 가장 기초적인 원가입니다.
258. 가공원가(전환원가)는 직접노무원가와 제조간접원가의 합이다. / 답: O / 설명: 재료를 제품으로 변환시키는 데 드는 원가입니다.
259. 제조간접원가는 직접재료원가와 직접노무원가를 제외한 모든 제조원가를 포함한다. / 답: O / 설명: 간접재료비, 간접노무비, 제조경비 등의 합입니다.
260. 제품원가는 자산화될 수 있는 원가이며, 기간원가는 발생 즉시 비용으로 처리된다. / 답: O / 설명: 재고자산 가액 포함 여부에 따른 분류입니다.
261. 제조활동과 관련이 없는 판매비와 관리비는 항상 기간원가로 분류된다. / 답: O / 설명: 비제조원가이기 때문입니다.
262. 관련범위 내에서 조업도가 변하더라도 원가 행태가 일정하게 유지된다고 가정한다. / 답: O / 설명: 원가 추정의 기본 전제입니다.

264. 단계원가(준고정원가)는 특정 범위 내에서는 일정하지만 범위를 벗어나면 계단식으로 증가한다. / 답: O / 설명: 감독관 급여 등이 대표적인 사례입니다.

501. 원가관리회계는 기업의 내부 경영자에게 경제적 의사결정에 필요한 정보를 제공하는 것이 주된 목적이다. / 답: O / 설명: 관리회계의 정의와 목적입니다.

503. 직접원가는 특정 원가대상에 경제적으로 추적이 가능한 원가이며, 직접재료비와 직접노무비가 대표적이다. / 답: O / 설명: 추적 가능성에 따른 분류입니다.
504. 고정원가는 조업도가 증가하더라도 총원가는 일정하지만 단위당 원가는 조업도에 반비례하여 감소한다. / 답: O / 설명: 고정비의 조업도별 행태입니다.
505. 변동원가는 조업도가 증가함에 따라 총원가는 정비례하여 증가하며 단위당 원가는 일정하다. / 답: O / 설명: 변동비의 조업도별 행태입니다.
506. 매몰원가는 이미 발생한 원가로서 의사결정 대안 간에 차이가 나지 않으므로 비관련원가에 해당한다. / 답: O / 설명: 의사결정 시 배제해야 할 원가입니다.
