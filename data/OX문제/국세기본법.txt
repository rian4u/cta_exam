### [제1파트] 국세기본법 OX 문제 100선 (총칙 ~ 조세채권의 보전)

1. 국세기본법은 국세에 관한 기본적 사항과 공통적인 사항을 정하여 세법 관계를 명확하게 함을 목적으로 한다. / 답: O / 설명: 국세기본법의 제1조 목적 규정입니다.
2. 국세기본법은 다른 세법에 우선하여 적용되므로, 개별 세법에서 국세기본법과 다른 규정을 두고 있더라도 국세기본법이 항상 우선한다. / 답: X / 설명: 개별 세법에서 국세기본법에 대한 특례 규정을 두고 있는 경우에는 그 세법이 정하는 바에 따릅니다.
3. 가산세는 해당 의무가 규정된 세법의 해당 국세의 세목으로 한다. / 답: O / 설명: 가산세는 국세의 세목으로 포함됩니다.
4. 해당 국세를 감면하는 경우 가산세도 함께 감면하는 것이 원칙이다. / 답: X / 설명: 가산세는 국세 감면의 대상에 포함되지 않습니다.
5. 세법의 해석이 확립된 후 새로운 해석에 의하여 소급하여 과세하지 아니한다는 원칙을 '소급과세금지의 원칙'이라 한다. / 답: O / 설명: 납세자의 신뢰보호를 위한 원칙입니다.
6. 기간의 계산에 있어서 국세기본법 또는 세법에 특별한 규정이 있는 것을 제외하고는 민법을 준용한다. / 답: O / 설명: 기간 계산의 일반 원칙입니다.
7. 납세자가 세법에 따른 기한까지 신고를 마쳤으나 우편으로 제출한 경우, 그 통지서가 도달한 날을 신고일로 본다. / 답: X / 설명: 우편으로 제출한 서류는 통신날짜도장이 찍힌 날(발신주의)에 신고된 것으로 봅니다.
8. 공휴일·토요일 또는 근로자의 날인 경우 그 다음 날을 기한으로 한다. / 답: O / 설명: 기한의 특례 규정입니다.
9. 국세청장이 정하는 기준에 해당하는 고액·상습체납자의 명단은 공개할 수 있다. / 답: O / 설명: 조세 정의 실현을 위한 명단 공개 제도입니다.
10. 서류의 송달은 교부, 우편 또는 전자송달의 방법으로 한다. / 답: O / 설명: 국세기본법상 서류 송달의 원칙적인 방법입니다.
11. 연대납세의무자에게 서류를 송달할 때 대표자가 없는 경우에는 그 중 1인에게만 송달하면 전원에게 효력이 있다. / 답: X / 설명: 대표자가 없는 경우 연대납세의무자 각자에게 서류를 송달해야 합니다. (단, 고지·독촉은 각자에게 송달)
12. 서류를 송달받아야 할 자가 송달받기를 거부하지 아니하면 송달할 장소가 아닌 곳에서 교부할 수 있다. / 답: O / 설명: 이를 '유치송달'이 아닌 '유효한 교부'로 인정합니다.
13. 주소나 영업소가 국외에 있어 송달이 곤란한 경우 공고한 날부터 14일이 지나면 송달된 것으로 본다. / 답: O / 설명: 공시송달의 효력 발생 시기입니다.
14. 법인격 없는 단체 중 이익의 분배 방법이 정해져 있지 않은 단체는 법인으로 보지 않는다. / 답: X / 설명: 당연법인 요건을 갖추거나 승인을 받은 경우에는 법인으로 보아 국세기본법을 적용합니다.
15. 법인으로 보는 단체 외의 단체는 거주자 또는 비거주자로 보아 국세에 관한 규정을 적용한다. / 답: O / 설명: 단체의 성격에 따라 개인으로 취급합니다.
16. 상속이 개시된 경우 그 상속인은 피상속인에게 부과되거나 피상속인이 납부할 국세를 상속으로 받은 재산의 한도 내에서 납부할 의무를 진다. / 답: O / 설명: 상속에 따른 납세의무의 승계 규정입니다.
17. 법인이 합병한 경우 합병 후 존속하는 법인은 합병으로 소멸된 법인의 납세의무를 승계한다. / 답: O / 설명: 법인 합병에 따른 무한 승계입니다.
18. 연대납세의무자 중 1인에 대한 국세의 징수권 소멸시효 완성은 다른 연대납세의무자에게 영향을 주지 않는다. / 답: X / 설명: 소멸시효 완성의 효력은 연대납세의무자 전원에게 미칩니다.
19. 국세의 납세의무는 세법이 정하는 과세요건이 충족되는 때에 성립한다. / 답: O / 설명: 납세의무 성립의 시점입니다.
20. 부가가치세의 납세의무 성립 시기는 과세기간이 끝나는 때이다. / 답: O / 설명: 기간 단위 과세 세목의 특징입니다.
21. 소득세와 법인세는 소득을 얻는 때마다 납세의무가 성립한다. / 답: X / 설명: 과세기간이 끝나는 때에 성립합니다.
22. 가산세는 가산할 국세의 납세의무가 성립하는 때에 함께 성립한다. / 답: O / 설명: 종된 납세의무의 성질을 가집니다.
23. 원천징수하는 소득세는 소득금액을 지급하는 때에 납세의무가 성립한다. / 답: O / 설명: 수시 성립하는 세목입니다.
24. 납세의무자가 과세표준과 세액을 정부에 신고하는 때에 확정되는 국세를 '신고납세제도'라 한다. / 답: O / 설명: 소득세, 법인세, 부가가치세 등이 해당합니다.
25. 인지세는 납세의무가 성립하는 때에 특별한 절차 없이 세액이 확정된다. / 답: O / 설명: 자동확정 국세입니다.
26. 정부가 과세표준과 세액을 결정하는 때에 확정되는 국세에는 상속세와 증여세가 포함된다. / 답: O / 설명: 정부부과과세 제도입니다.
27. 국세채권의 소멸시효는 일반적으로 5년이며, 5억원 이상의 국세는 10년이다. / 답: O / 설명: 국세 징수권의 소멸시효 기간입니다.
28. 납세고지, 독촉, 압류는 소멸시효를 중단시킨다. / 답: O / 설명: 시효 중단 사유입니다.
29. 소멸시효가 중단된 경우, 중단 사유가 종료된 때부터 시효가 새로이 진행된다. / 답: O / 설명: 중단과 정지의 차이입니다.
30. 국세를 포탈하거나 환급·공제받은 경우의 국세 부과 제척기간은 원칙적으로 10년이다. / 답: O / 설명: 부정행위에 대한 엄격한 적용입니다.
31. 상속세와 증여세의 부과 제척기간은 원칙적으로 10년이며, 부정행위 시 15년이다. / 답: O / 설명: 다른 국세보다 긴 제척기간을 가집니다.
32. 제척기간에는 소멸시효와 달리 중단이나 정지 제도가 없다. / 답: O / 설명: 제척기간은 법률관계의 조속한 확정을 위한 강제 기간입니다.
33. 수정신고를 한 경우 관할 세무서장은 신고일부터 2개월 이내에 세액을 확정해야 한다. / 답: X / 설명: 수정신고는 신고납세제도 국세의 경우 신고 시 즉시 확정되는 것이 원칙입니다. (정부부과 국세는 제외)
34. 경정청구는 과세표준 신고서를 법정기한 내에 제출한 자가 5년 이내에 할 수 있다. / 답: O / 설명: 납세자의 과다납부 구제 절차입니다.
35. 세무서장은 경정청구를 받은 날부터 2개월 이내에 결과를 통지해야 한다. / 답: O / 설명: 경정청구 처리 기한입니다.
36. 납세자가 국세의 납부기한까지 납부하지 아니한 경우 부과되는 것을 '가산세'라 한다. / 답: O / 설명: 현재는 미납부 가산세로 통합되어 운영됩니다.
37. 국세체납처분비는 국세, 가산금에 우선하여 징수한다. / 답: O / 설명: 징수 비용의 우선순위입니다.
38. 법정기일 전에 설정된 전세권에 의해 담보된 채권은 국세에 우선한다. / 답: O / 설명: 국세우선권의 예외(설정일 기준)입니다.
39. 주택임차보증금 중 일정액(소액보증금)은 법정기일과 관계없이 국세보다 우선하여 변제된다. / 답: O / 설명: 사회적 약자 보호를 위한 최우선 변제권입니다.
40. 국세의 법정기일이란 과세표준과 세액의 신고일 또는 고지서 발송일을 의미한다. / 답: O / 설명: 우선순위 판단의 기준 시점입니다.
41. 제2차 납세의무는 주된 납세의무자의 재산으로 국세를 완납할 수 없는 경우에 발생한다. / 답: O / 설명: 보충적 납세의무입니다.
42. 청산인은 분배한 재산의 가액을 한도로 제2차 납세의무를 진다. / 답: O / 설명: 청산인 등의 책임 범위입니다.
43. 무한책임사원은 법인의 재산으로 국세를 완납할 수 없는 경우 부족액에 대해 제2차 납세의무를 진다. / 답: O / 설명: 인적회사의 책임 구조입니다.
44. 과점주주 중 경영을 사실상 지배하는 자는 제2차 납세의무를 질 수 있다. / 답: O / 설명: 실질적 지배주주에 대한 책임 추궁입니다.
45. 사업을 양수한 자는 양도일 이전에 확정된 국세를 양수한 재산 가액 한도로 납부할 의무를 진다. / 답: O / 설명: 사업양수인의 제2차 납세의무입니다.
46. 국세환급금은 환급 결정일로부터 30일 이내에 지급해야 한다. / 답: X / 설명: 지체 없이 환급하여야 합니다.
47. 납세자가 체납한 국세가 있는 경우 국세환급금을 체납액에 충당할 수 있다. / 답: O / 설명: 환급금의 직권 충당 제도입니다.
48. 국세환급가산금은 국세환급금을 지급할 때 이자 성격으로 추가하여 지급하는 금액이다. / 답: O / 설명: 납세자의 금전적 손실 보전입니다.
49. 국세환급금에 관한 권리는 5년간 행사하지 아니하면 소멸시효가 완성된다. / 답: O / 설명: 납세자의 환급청구권 시효입니다.
50. 신의성실의 원칙은 납세자에게만 적용되며 세무공무원에게는 적용되지 않는다. / 답: X / 설명: 양측 모두에게 적용되는 대원칙입니다.
51. 근거과세의 원칙에 따라 장부와 증빙자료가 없는 경우에도 추정하여 과세하는 것이 원칙이다. / 답: X / 설명: 장부와 증빙에 의한 과세가 원칙이며, 추계과세는 극히 예외적인 경우에만 허용됩니다.
52. 조세특례제한법에 따른 조세감면 규정은 엄격하게 해석해야 하며 유추해석은 금지된다. / 답: O / 설명: 법적 안정성과 예측 가능성을 위함입니다.
53. 국세기본법상 '세무조사'는 국세의 과세표준과 세액을 결정하거나 경정하기 위하여 질문하거나 장부 등을 검사하는 활동이다. / 답: O / 설명: 세무조사의 정의입니다.
54. 세무공무원은 납세자가 성실하며 제출한 신고서가 진실한 것으로 생각하고 업무를 처리해야 한다. / 답: O / 설명: 성실성 추정의 원칙입니다.
55. 납세자는 세무조사를 받는 경우 세무사나 변호사를 조력자로 참여시킬 수 있다. / 답: O / 설명: 납세자의 방어권 보장입니다.
56. 세무조사 사전통지는 조사 개시 15일 전까지 해야 한다. / 답: O / 설명: 조사 준비를 위한 사전 알림입니다.
57. 증거인멸의 우려가 있는 경우 사전통지 없이 세무조사를 시작할 수 있다. / 답: O / 설명: 사전통지의 예외 사유입니다.
58. 동일한 과세기간에 대하여 2회 이상 세무조사를 하는 것은 원칙적으로 금지된다. / 답: O / 설명: 중복조사 금지의 원칙입니다.
59. 세무조사 결과 통지는 조사를 마친 날부터 20일 이내에 해야 한다. / 답: O / 설명: 조사의 신속한 마무리와 통지 의무입니다.
60. 과세전적부심사청구는 고지서를 받기 전 단계에서 하는 구제 절차이다. / 답: O / 설명: 사전적 권리 구제 제도입니다.
61. 과세전적부심사청구는 통지를 받은 날부터 30일 이내에 해야 한다. / 답: O / 설명: 청구 기간입니다.
62. 세법에 따른 처분으로서 위법하거나 부당한 처분을 받은 자는 이의신청, 심사청구, 심판청구를 할 수 있다. / 답: O / 설명: 조세 불복 제도입니다.
63. 심사청구와 심판청구는 중복하여 제기할 수 없다. / 답: O / 설명: 선택적 중복 금지 원칙입니다.
64. 이의신청은 처분이 있음을 안 날부터 90일 이내에 해야 한다. / 답: O / 설명: 불복 청구 기간의 일반 원칙입니다.
65. 심판청구에 대한 결정은 청구를 받은 날부터 90일 이내에 해야 한다. / 답: O / 설명: 결정 기한입니다.
