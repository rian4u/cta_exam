# Roadmap

## 우선순위 1

- 로그인 기능
  - 현재는 기기 기반(`device-id`) 상태 저장 중심
  - 계정 기반으로 오답노트, 코멘트, 즐겨찾기 동기화 필요
  - 최소 요구:
    - 회원 식별
    - 로그인 유지
    - 사용자별 데이터 분리

- 두문자관리
  - 문제/과목 단위 두문자 메모 저장
  - 검색/필터링
  - 오답관리와 연결된 복습 흐름
  - 최소 요구:
    - 저장/수정/삭제
    - 과목별 조회
    - 키워드 검색

- 게시판 기능 고도화
  - 현재는 시범용 입력/표시 수준
  - 실제 운영형 게시판으로 전환 필요
  - 최소 요구:
    - 질문/답변 수정/숨김
    - 관리자 기능
    - 신고/스팸 대응

## 우선순위 2

- 데이터 검증 자동화
  - 풀이 답안 vs 배포답안 차이 검토
  - 법 개정 이슈 분리 표시
- 연도별 공개 정책 정리
  - 2024~2022 검수 완료 후 단계적 오픈
- 모바일 UI 추가 보정
  - 게임 외 일반 화면 터치 UX 재점검

## 우선순위 3

- 운영용 관리자 화면
  - 공지 작성
  - 게시판 관리
  - 사용자 신고 처리
- 배포 운영 문서 정리
  - 백업/복구
  - 배포 전 체크리스트
