# config

로컬 설정 파일을 두는 폴더입니다.

## 사용 예시

- `gemini_api_key.txt`
  - LLM 보조 스크립트에서 사용할 Gemini API 키
- `openai_api_key.txt`
  - 향후 OpenAI 연동을 추가할 경우 사용할 키 파일

## 원칙

- API 키는 코드에 하드코딩하지 않습니다.
- 한 줄짜리 텍스트 파일로 분리 저장합니다.
- 이 폴더의 실제 키 파일은 Git에 올리지 않는 것이 원칙입니다.

## 주의

- 배포 서비스 런타임에는 대부분 필요하지 않습니다.
- 데이터 구축/검증 스크립트 실행 시에만 필요한 경우가 많습니다.
