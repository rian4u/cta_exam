# 2025 문제 DB 적재

## 실행

```powershell
python scripts/load_2025_questions.py --data-dir data/2025 --db-path data/questions.db --year 2025
```

## 적재 대상

- 1교시 PDF: `재정학(1~40)`, `세법학개론(41~80)`
- 2교시 PDF: `회계학개론(1~40, 중복 1회만)`, 선택법 `상법/민법/행정소송법(41~80)`
- `~~풀이.txt`: `답`, `해설`
- `실제정답.txt`: `답_배포`

## 문제 출력(렌더링) 방식

- `문제지문`, `보기_1~보기_5`는 PDF에서 추출된 공백/줄바꿈(`\n`)을 보존하여 저장된다.
- 웹 출력 시 `white-space: pre-wrap;`으로 렌더링하면 표/계산식 줄 구조를 그대로 표시할 수 있다.
- 콘솔 출력 시에도 원문 줄바꿈을 유지해 출력한다.
- `렌더_마크업` 컬럼에 문제/보기의 HTML 렌더 정보(`stem_html`, `options_html`)를 함께 저장한다.
- 표/박스는 HTML 구조(`rich-table`, `rich-box`)로 저장되며, API 응답 시 그대로 전달되어 웹앱에서 원형에 가깝게 표시된다.
- 수식 특수문자(PUA)는 정규화되어 저장되고, 웹앱에서 MathJax를 통해 LaTeX 수식 표기가 함께 렌더링된다.
